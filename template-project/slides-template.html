Qualidade de código

Lint, formatação e hook de commit

  • ESLint (npm run lint) aponta erros e maus hábitos.
  • Prettier (npm run format) mantém um padrão único.
  • Husky roda os dois no pre-commit, em qualquer branch.

Código com erro de lint ou fora do padrão não entra no histórico.
